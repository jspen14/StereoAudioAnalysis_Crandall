Instructions (Written for Ubuntu 16.04)

1. Plug both microphones into the computer
2. Open System Settings > Sound > Input
  a. Ensure both microphones show up, have "Input volume" set at 100%,
      and are picking sounds.
3. Run micSetup.py (Finds indices of USB PnP Audio Devices)
  a. Note: If using different microphones, this script work as expected.
